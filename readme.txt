A wordpress plugin that helps to capture Contact Form 7 submissions

== Description ==

Efthakhar CF7 Database plugin help you to save Contact Form 7 submissions in your wordpress database and view the submissions from wordpress admin area.

= FEATURES =
* save form submissions of contact from 7
* view submissions for specific form 
* view submissions only for certain form fields
* delete selected submissions in one click
* export selected submissions or all submissions of a form as csv file 
* persist deleted or modified fields data even after editing cf7 form


== Installation ==

Easy way:
1. Go to **WP Admin - Plugins - Add New**
2. Search for "Efthakhar CF7 Database"
3. Install the plugin and then Activate it

or Manual way:
1. Download the plugin zip file
2. Extract it
3. Upload the plugin directory to **wp-content/plugins**
4. Activate **Efthakhar CF7 Database** plugin from **WP Admin - Plugins**

== Frequently Asked Questions ==

= Does it need any plugin other than Contact Form 7 for integration? =

No plugins are needed except Contact Form 7.
